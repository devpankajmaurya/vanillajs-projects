# Vanilla JS Projects

This repository contains a collection of projects built using **HTML, CSS, Vanilla JavaScript, and Node.js**.
The goal of this repository is to practice **core JavaScript concepts, DOM manipulation, and full-stack web development** by building real projects.

---

## 🚀 Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* DOM Manipulation
* Node.js *(for backend in future projects)*

---

## 📂 Projects

### 1. Product Image Gallery

A product gallery similar to ecommerce websites.

**Features**

* Thumbnail image switching
* Previous / Next navigation
* Keyboard navigation (Arrow keys)
* Active thumbnail highlighting
* Image zoom on hover
